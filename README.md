# To Do List Web App

Simple browser-based to‑do list that persists tasks locally and lets you add, mark complete, and delete items with a clean UI.

## Project structure
- `index.html` — page layout and root containers.
- `style.css` — styling for the list, controls, and responsive layout.
- `script.js` — client-side logic (task creation, completion toggles, deletion, and local storage).

## Running locally
1. Open `index.html` directly in a modern browser (Chrome, Firefox, Edge).
2. Your tasks are stored in `localStorage`; they stay on this device until you clear them.

## Development notes
- No build step or dependencies required; everything runs in the browser.
- If you edit scripts or styles, refresh the page to see changes.
